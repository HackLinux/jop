Configuration

This package provides a generic configuration component.

The configuration consists of a set of available options, and the actual configuration data.

Packages or classes, which need external configuration, provide a collection of available options the client may provide. Those options are explicitly added to the configuration manager at application startup.

The configuration data is loaded using system properties, property files and/or command line arguments. The classes which need to obtain a configuration value ask the configuration manager on demand.

Additionally, this component provides help messages, logger configuration and parsing / syntax checking of various option formats
